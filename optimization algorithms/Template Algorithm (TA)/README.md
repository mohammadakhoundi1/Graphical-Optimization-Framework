# Template Algorithm (TA)

Copy this folder when adding a new algorithm.

## Steps
1. Duplicate this folder into the correct category.
2. Rename the folder to: `Full Name (ACRONYM)`
3. Fill in:
   - `README_ALGORITHM.md` (then rename it to `README.md`)
   - `CITATION.md`
   - `METADATA.json`
4. Replace `Algorithm.m` with the actual implementation (or add multiple `.m` files).
5. Add any additional assets under `assets/`.

## Notes
- Do not remove provenance fields.
- If you modified upstream code, document what changed and why.
